# Excel数据筛选工具 使用说明

## 工具简介
这是一个无需安装Python环境的Excel数据筛选工具，支持多条件筛选数据，并将结果追加到指定Sheet中。

## 功能特点
- 支持打开xlsx/xls格式的Excel文件
- 自动解析表头，可视化选择筛选列
- 支持多种运算符：等于、不等于、大于、小于、大于等于、小于等于、包含、不包含
- 支持多条件组合，条件之间可设置"与/或"逻辑关系
- 筛选结果可追加到现有Sheet（保留原有数据）或创建新Sheet
- 操作简单，界面友好

## 使用方法

### 第一步：选择文件
1. 点击"浏览"按钮
2. 选择要处理的Excel文件（.xlsx或.xls格式）

### 第二步：选择Sheet
1. 从下拉菜单中选择要处理的工作表
2. 程序会自动解析表头数据

### 第三步：添加筛选条件
1. 点击"添加条件"按钮
2. 在弹出的对话框中：
   - 选择要筛选的列名
   - 选择运算符（=, !=, >=, <=, >, <, 包含, 不包含）
   - 输入比较值
   - 选择与下一条件的关系（与/或）
3. 点击"确定"添加条件
4. 可添加多个条件，条件会按顺序执行

### 第四步：设置目标Sheet
1. 在"目标Sheet名称"输入框中输入目标Sheet名称
2. 如果该Sheet已存在，数据将追加到该Sheet（不会覆盖原有数据）
3. 如果Sheet不存在，将创建新Sheet
4. 留空则默认创建名为"筛选结果"的新Sheet

### 第五步：执行筛选
1. 点击"执行筛选"按钮
2. 程序会按照设定的条件筛选数据
3. 完成后会显示找到的数据行数

### 第六步：保存结果
1. 点击"保存结果"按钮
2. 选择保存位置和文件名
3. 文件保存为.xlsx格式

## 常见使用场景

### 场景1：筛选N列到U列中有0值的数据
1. 添加条件：列=N, 运算符=等于, 值=0, 逻辑=或
2. 添加条件：列=O, 运算符=等于, 值=0, 逻辑=或
3. ...（重复添加N到U列的每个列）
4. 最后一个条件设置逻辑为"无"
5. 执行筛选

### 场景2：筛选特定条件的数据
1. 添加条件：列=状态, 运算符=等于, 值=完成, 逻辑=与
2. 添加条件：列=金额, 运算符=大于等于, 值=1000, 逻辑=无
3. 执行筛选

## 条件逻辑说明
- **与 (AND)**: 必须同时满足当前条件和下一个条件
- **或 (OR)**: 满足当前条件或下一个条件即可
- **无 (最后一个条件)**: 表示这是最后一个条件

例如：
- 条件1(列=A, 值=1, 逻辑=与) + 条件2(列=B, 值=2, 逻辑=无)
  表示: A=1 且 B=2

- 条件1(列=A, 值=1, 逻辑=或) + 条件2(列=B, 值=2, 逻辑=无)
  表示: A=1 或 B=2

## 打包说明

### Windows系统打包
1. 确保已安装Python 3.8或更高版本
2. 双击运行 `build.bat`
3. 等待打包完成
4. 在 `dist` 目录下找到 `Excel数据筛选工具.exe`

### Mac/Linux系统打包
1. 确保已安装Python 3.8或更高版本
2. 在终端运行：`bash build.sh`
3. 等待打包完成
4. 在 `dist` 目录下找到可执行文件

## 注意事项
1. 原文件会被修改，建议先备份
2. 保存结果时会生成新文件，不会覆盖原文件
3. 大文件处理可能需要较长时间，请耐心等待
4. 如果遇到程序无法打开，请检查是否有杀毒软件拦截

## 系统要求
- Windows 7/10/11 或 macOS 10.13+ 或 Linux
- 无需安装Python或其他依赖
