# 🚀 最快速获得exe的方法

## 方法：使用GitHub自动打包（推荐⭐⭐⭐⭐⭐）

**只需5分钟，不需要安装任何东西！**

### 📝 操作步骤

#### 1️⃣ 上传到GitHub（2分钟）
- 打开 https://github.com/new
- 创建一个新仓库（Public或Private都可以）
- 把下面的文件拖拽上传：
  ```
  ✅ excel_filter_tool.py
  ✅ .github/workflows/build.yml
  ```
- 点击 "Commit changes"

#### 2️⃣ 自动构建（1分钟）
- 在仓库页面点击 **"Actions"** 标签
- 点击左侧的 **"Build Windows EXE"**
- 点击右侧的 **"Run workflow"** 按钮
- 再点击绿色的 **"Run workflow"** 确认

#### 3️⃣ 下载exe（2分钟）
- 等待约1-2分钟，刷新页面
- 看到绿色的 ✓ 表示构建成功
- 点击进入构建任务
- 滚动到页面底部 **"Artifacts"** 区域
- 下载 `Excel数据筛选工具-Windows`
- 解压zip文件，里面就是 `Excel数据筛选工具.exe`

### 🎉 完成！

现在可以把exe文件拷贝到U盘，在任何Windows电脑上使用！

## 🔧 如果GitHub方法不行

**备选方案：** 使用Windows电脑

1. 把文件复制到Windows电脑
2. 双击 `build.bat`
3. 等待完成
4. 在 `dist` 文件夹得到exe
